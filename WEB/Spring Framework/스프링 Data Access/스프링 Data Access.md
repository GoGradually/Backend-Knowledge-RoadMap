# 스프링 Data Access란?
- JDBC
	- DB 연결 및 쿼리 실행
- ORM
	- JPA, Hibernate 등의 ORM 과 통합 제공
- 트랜잭션 관리
	- 선언적 트랜잭션 관리
- OXM
	- 객체와 XML 간 매핑
- JMS
	- Java Message Service 를 통한 메시징 기능
