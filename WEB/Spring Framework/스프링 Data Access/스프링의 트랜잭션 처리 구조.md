- 트랜잭션이 고려하는 것
    
    - 커넥션 유지
        
        - 트랜잭션은 세션이 진행하는 거임
            
    - 서비스 계층에서 커넥션 선택 후, 해당 커넥션을 계속 사용해야 함
        
        - 메소드 단위로 커넥션을 진행하면 서비스 계층의 비즈니스 로직에 트랜잭션이 걸리지 않음![[Pasted image 20241215191911.png]]
            
    - 커넥션 사용 시
        
        - setAutoCommit(false) 후
            
        - 잘 사용한 뒤
            
        - setAutoCommit(true) 로 재전환
        - 다른 메소드들이 쓸 때, 오토커밋이 false로 되어있지 않도록
