- Part-Whole 관계에서, **완전한 소유** 개념이 들어간 관계
	- 가장 강한 has-a 관계
	- Part 는 공유되지 않는다.
- 상위 타입이 하위 타입의 **생명주기를 완전히 관리**한다.
	- 상위 타입이 하위 타입의 생명주기를 결정할 수 있다.
		- 자신이 파괴될 때 하위 타입을 전부 파괴할 수도 있고
		- 자신이 소유한 Part 를 다른 Whole 에게 **건네줄 수 있다**.
