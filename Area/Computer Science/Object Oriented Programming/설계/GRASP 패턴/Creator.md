- 객체 생성의 책임을 할당하는 패턴
- 다음의 경우 B객체를 생성할 책임을 A클래스에 할당
    - A가 B를 포함하거나 참조
    - A가 B를 기록
    - A가 B를 밀접하게 사용
    - A가 B의 초기화 데이터를 가짐
- 왜?
	- **의존성!**
	- 이미 B가 A에 **구조적 관계**를 갖고 있으므로
		- 해당 객체를 직접 생성한다고, 추가적인 관계가 생성되지 않는다!
